This project analyzes the 2019 New York City Airbnb dataset to explore pricing patterns, neighborhood trends, room type differences, and host behavior. The project includes exploratory data analysis, statistical testing, regression modeling, classification, clustering, and Random Forest regression to better understand factors associated with Airbnb listing prices.

**Tools Used:** Python, pandas, NumPy, matplotlib, seaborn, scipy, pingouin, scikit-learn

**Skills Shown:** Data cleaning, exploratory data analysis, feature transformation, hypothesis testing, ANOVA, regression modeling, classification, clustering, Random Forest modeling, model evaluation
