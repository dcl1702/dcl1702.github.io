excerpt: "A machine learning project analyzing how Spotify audio features such as danceability, loudness, acousticness, tempo, and speechiness relate to song popularity."
collection: portfolio
---

This project uses the Spotify Tracks Dataset to analyze whether audio features can help explain and predict track popularity. The analysis compares baseline models with a Random Forest Regressor, evaluates model performance, and uses feature importance and partial dependence plots to interpret which audio characteristics are associated with higher predicted popularity.

**Tools Used:** Python, pandas, NumPy, matplotlib, seaborn, scikit-learn

**Skills Shown:** Exploratory data analysis, regression modeling, Random Forest modeling, model evaluation, feature importance analysis, partial dependence plots, business interpretation
