This project builds a predictive data mining model to identify which bank customers are most likely to accept a personal loan offer. The analysis includes exploratory analysis, data cleaning, feature engineering, log transformation, scaling, categorical encoding, and classification modeling.

**Tools Used:** Python, pandas, scikit-learn, matplotlib, seaborn

**Skills Shown:** Data preprocessing, feature engineering, classification modeling, logistic regression, decision tree modeling, model evaluation
