This project uses a health dataset to examine how demographic, lifestyle, and medical factors relate to type 2 diabetes risk. 
The analysis includes exploratory data analysis, data preprocessing, classification modeling, and clustering to identify behavioral risk profiles.

**Tools Used:** Python, pandas, scikit-learn, matplotlib, seaborn

**Skills Shown:** Data cleaning, exploratory analysis, classification modeling, random forest, logistic regression, clustering
