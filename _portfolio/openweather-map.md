This project is a Python weather station application that connects to the OpenWeatherMap API to retrieve current weather data. The user can enter a U.S. city and state or ZIP code, and the program returns a readable weather report with temperature, conditions, cloud coverage, pressure, and humidity.

**Tools Used:** Python, requests, OpenWeatherMap API, JSON

**Skills Shown:** API integration, JSON parsing, user input handling, exception handling, command-line application design
