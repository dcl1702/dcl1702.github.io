This project groups multiple R-based statistical analysis reports into one portfolio project. The analyses include a paired t-test on candy consumption, correlation and confidence interval analysis using travel insurance data, ANOVA model comparison for net sales, and housing price regression modeling.

**Tools Used:** R, R Markdown, ggplot2, corrplot, AICcmodavg, statistical modeling

**Skills Shown:** Hypothesis testing, correlation analysis, covariance, confidence intervals, ANOVA, AIC model comparison, regression modeling, statistical interpretation
