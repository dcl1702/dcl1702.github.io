title: "Earnings Call Evasive Signal Detection"
excerpt: "A GenAI text analysis project that screens earnings call transcript excerpts for evasive communication patterns such as hedging, vague language, partial explanations, and future-focused redirection."
collection: portfolio
---

This project uses generative AI to analyze earnings call transcript excerpts and identify possible evasive communication signals. The model classifies transcript excerpts by evasiveness level, score, main signal type, evidence quote, reasoning, and confidence level to help analysts find sections that may need closer review.

**Tools Used:** Python, OpenAI API, SQLite, pandas, matplotlib, seaborn, JSON

**Skills Shown:** GenAI text analysis, prompt engineering, transcript processing, structured JSON output, SQLite storage, data visualization, model output analysis, financial communication analysis
