This project combines three data sources to examine relationships between national socioeconomic indicators, university presence, and literacy rates across countries. The project includes flat file cleaning, web scraping, API data collection, data standardization, SQLite database integration, table joins, and visual analysis.

**Tools Used:** Python, pandas, BeautifulSoup, requests, SQLite, matplotlib, seaborn

**Skills Shown:** Data wrangling, API integration, web scraping, data cleaning, database loading, SQL joins, data visualization, multi-source analysis
