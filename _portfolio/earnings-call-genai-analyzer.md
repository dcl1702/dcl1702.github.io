This project is a GenAI-powered Streamlit application for analyzing earnings call transcripts. Users can upload a PDF transcript or paste transcript text, and the app returns structured financial analysis including sentiment, summary, positive factors, explicit risks, implicit risks, and key financial concerns.

**Tools Used:** Python, Streamlit, OpenAI API, pypdf, JSON, dotenv

**Skills Shown:** GenAI application development, prompt engineering, fine-tuned model usage, PDF text extraction, structured JSON output, Streamlit UI design, financial text analysis
